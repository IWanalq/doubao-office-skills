# 对话框深度回复格式

对话框回复是默认成品,不是 HTML 的短入口。用户不打开 HTML 也应能读懂核心判断、关键依据、关键增量信息、为什么重要、市场影响链、主要分歧、验证/证伪、来源和信息边界。普通对话按 `references/dialog-delivery.md` 直接输出;已有 `briefing.json` 或外部工作流交付时,由 `scripts/render_dialog_brief.py` 从完整 JSON 生成。

对话框和 HTML 必须同构:顶层标题必须使用「核心观点 / 有哪些关键增量信息？ / 这些信息为什么重要？ / 上述信息如何影响市场？ / 各方有哪些值得关注的观点？ / 哪些信号会验证核心观点？哪些信号代表观点证伪？ / 信息来源」。不要把外部模型生成的内容改成「一、政策背景 / 二、当前立场 / 三、市场共识 / 四、监测表 / 五、核心结论」这类自拟报告目录。

## 结构

1. **核心判断**(`dialog_brief.verdict`):一句话直接回答用户问题。
2. **完整摘要**(`dialog_brief.executive_summary`):把事实变化、市场含义、限制条件和分歧讲顺。
3. **为什么这么看**(`dialog_brief.key_points`):4-6 条,覆盖事实变化、市场含义、分歧点、后续验证,每条带 `refs`。
4. **有哪些关键增量信息？**(`news_items`):2-4 条新增事实,用分组要点展示,不使用表格。
5. **这些信息为什么重要？**(`insight.why_now / what_changed / consensus_vs_delta`):讲清真正变化和预期差。
6. **上述信息如何影响市场？**(`visuals.impact_chain / impact_facts`):把事实翻译成市场变量和传导路径。
7. **各方有哪些值得关注的观点？**(`interpretation`):按来源角色区分官方事实、市场定价、机构观点和反向证据。
8. **哪些信号会验证核心观点？哪些信号代表观点证伪？**(`verification.confirm / invalidate`):分别写确认主线和削弱或推翻主线的信号。
9. **信息来源**:列出被引用的核心来源标题、来源名、日期和原文链接。
10. **边界**(`dialog_brief.risk_boundary`)和 **HTML 选项**:默认结尾必须逐字写「需要为你生成一份精美的热点报告么？」,不得改写、润色或补写「HTML 格式」「解读简报」等词;只有已生成 HTML 时才给「详细 HTML 简报: <HTML 链接>」。

## 版式

- 默认不用 Markdown 表格;窄屏可读性差,也容易把内容压成字段。
- 每个要点先给简短标签,再给解释。
- 来源编号要贴近被支撑的事实或判断。
- 来源账本必须保留原文链接。

## 语气红线

- 不含买卖建议、目标价、涨跌承诺或仓位建议。
- 不用「大概率」「必然」「确定」伪装确定性。
- 传闻不要当作事实写入;确需提及时写成争议点或官方回应。
- 不输出检索过程、交易日确认、执行记录、文件编辑记录或内部推理。

## 示例骨架

核心判断：CPO 短期不是缺长期需求,而是缺能让市场确认的订单和出货节奏。

这条主线现在的关键不在长期空间,而在近端兑现节奏。官方路线更新给了产业链新的时间锚,但券商和媒体更关注客户出货、供应链订单和排产是否同步确认;如果这些信号迟迟不出现,市场仍会把它当成预期交易而不是业绩验证。

为什么这么看：
- 事实变化：海外算力厂商更新产品推进路线,市场重新把供应链节奏放到核心位置。[1]
- 市场含义：长期算力需求仍是共识,但短期定价更看订单确认、排产上修和客户出货。[1][4]
- 分歧点：机构对近端放量更谨慎,分歧集中在兑现时间表。[3]
- 后续验证：接下来要看客户出货、供应链订单和排产信息是否连续出现公开确认。[4]

有哪些关键增量信息？
- **海外算力厂商公布产品推进路线**：官方公告披露产品推进路线和后续节点,市场把关注点转向供应链能否按节奏进入订单和出货验证。[1]

这些信息为什么重要？
- 现在重要：市场关注点已经从「长期有没有需求」转向「近端能不能放量」。
- 真正变化：机构更谨慎地评估订单、排产和客户出货节奏。
- 共识与分歧：长期算力需求仍是共识,分歧在近端兑现时间表。

上述信息如何影响市场？
- 起点：海外路线更新给供应链排产提供时间参照。[1]
- 传导：券商下修近端预期,市场重新评估业绩兑现速度。[3]
- 验证：客户出货和订单公开确认会决定主题能否从预期交易进入业绩验证。[4]

各方有哪些值得关注的观点？
- **官方事实**：路线更新确认中期方向,但没有直接给出 A 股相关公司的订单数据。[1]
- **反向证据**：机构谨慎点在近端放量节奏,这会压低短期业绩兑现的确定性。[3]

哪些信号会验证核心观点？哪些信号代表观点证伪？
确认主线：
- 客户或供应链披露出货确认。
- 上游订单和排产同步改善。
削弱或推翻：
- 订单继续延后。
- 机构继续下修近端收入假设。

信息来源
- [1] 海外算力厂商产品推进路线（官方公告, 2026-06-11 15:30）：https://example.com/official

边界：这里只做公开信息梳理和验证条件拆解,不构成投资建议。
